# syahril's test @kedatech
Front End Technical Test @Keda-Tech


First, cd to the directory:

```bash
cd test-kedatech
```
Then, install the required modules:
```bash
npm install
or
yarn install
```
Then, to run all the unit-test:
```bash
npm run test
or
yarn test
```
